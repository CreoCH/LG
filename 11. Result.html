Creo Simulate 교육

본 교육을 위해 아래 다운로드를 클릭해 교육용 CAD를 다운로드해 주시기 바랍니다.

이 파일은 Creo 5.0에서 제작하여 그 이하 버전에서는 정상적으로 열리지 않습니다.

다운로드

  • 결과 저장

    • 시뮬레이션에서 설정하고 저장한 조건들은 Creo parametric .PRT나 .ASM에 저장됩니다. 때문에 .STEP 등 기타 파일 형식으로 저장하면 설정 정보가 사라집니다. 시뮬레이션 분석 결과는 활성된 작업 디렉토리에 저장됩니다.

    결과에는 다음과 같은 정보가 포함됩니다.

  • 결과 확인 창

    • 홈 탭

    -. Opne : 저장된 결과를 불러옵니다. 결과로 불러올땐 저장된 폴더 자체를 선택해 주시면 됩니다. 폴더가 아래와 같은 아이콘으로 표시됩니다.

    -. 기타 Opne : 자주 사용하는 템플릿 형식이 있다면 저장해둔 후 결과 오픈 시 적용할 수 있습니다. 저장 형식은 .rwt입니다.

    혹은 디폴트값으로 지정해서 자동으로 열 수도 있습니다. 아래 그림에서 위쪽이 템플릿을 적용하려는 결과, 아래쪽이 템플릿 형식입니다.

    -. Edit : 현재 보고 있는 결과를 편집합니다.

    -. Copy : 현재 보고 있는 결과를 복사해 창을 띄웁니다.

    -. Dynamic query : 동적 퀴리는 그래픽 창 기준으로 특정값의 변경을 확인할 수 있습니다.

    -. Model Max Min : 그래픽창에서 맥스, 민의 위치를 포인트로 표시합니다.

    -. Measures : 사전 측정 정의 값, 요청한 측정 값을 한번에 확인합니다. 또한 측정한 값을 주석으로 생성할 수 있습니다.

    -. linearized stress : 포인트를 이용해 확인하고 싶은 부분의 응력을 확인합니다.

    보기 탭

    -. 방향 그룹 : 줌인, 줌아웃, 저장된 방향을 사용하거나 세부 방향을 설정할 수 있습니다.

    -. 모양새 그룹 : 결과를 보기 좋게 모양새를 변경합니다. 어셈블리라면 분해도 상태로 결과를 확인할 수 있습니다.

    -. 캡 및 절삭 서피스 그룹 : 단면뷰를 설정할 수 있습니다. 좌표계로부터의 갭으로 설정됩니다.

    ㅉ

    -. 애니메이션 그룹 : 애니메이션을 재생하거나 멈출수 있습니다. 재생 속도를 조정할 수 있습니다.

    -. 창 그룹 : 여러 결과 창을 확인할 때 창 정렬 방법과 창 가시성을 조정할 수 있습니다.

    형식 탭

    -. 범례 탭 : Tone 벨류나 색을 설정할 수 있습니다.

    -. 그래프 탭 : 현재 결과와 관련된 그래프 결과 창을 생성할 수 있습니다. 그래프도 미리 형식을 만들어서 사용할 수 있습니다.

    -. 주석 탭 : 메모, 지시선, 화살표 등 주석을 삽입할 수 있습니다.

  • 디스플레이 타입

    • 프린지

    프린지(Fringe)란 결과에 대한 값을 색상에 대비해 모델을 색상 영역으로 채운 결과입니다. 대부분의 시뮬레이션 결과값이 직관적인 표현을 위해 프린지 결과값을 사용하고 있습니다.

    -. Continuous Tone : 색상의 경계를 부드럽게 전환합니다.

    -. Averaged : 두 요소가 만나는 지점의 평균 값으로 수량을 표시합니다. 기본 옵션으로 플럭스와 온도 구배에 대해서만 체크 해제할 수 있습니다.

    -. Legend Levels : 모델에 표시할 다양한 색상 범위의 수를 설정합니다.(최소 2, 최대 15)

    -. Contour : 등고선 또는 등가면이 있는 모델을 표시합니다.

    -. Deformed : 모델을 변형된 상태로 표시합니다.

    -. Show Element Edges : 모델에서 요소의 가장자리를 표시합니다.

    -. Show Loads: 모델에 하중 아이콘을 표시합니다.

    -. Show Constraints : 모델에 구속 아이콘을 표시합니다.

    -. Show Constraints : 모델에 접합 요소를 표시합니다.

    -. Animate : 결과창을 애니메이트 할지 결정합니다.

    벡터

    힘의 방향성과 힘의 크기를 벡터로 표현합니다.

    -. Wireframe Vectors : 벡터를 와이어프레임 엔터티로 표시합니다.

    -. Shaded Vectors : 벡터를 음영 다각형 엔터티로 표시합니다.

    -. 그외 옵션 : 프린지 결과 옵션과 상동합니다.

    그래프

    선택한 수량을 그래프로 표시합니다. 그래프 결과 창 정의나 '형식 탭 > 그래프 그룹 > 편집'에서 X, Y축 표시, 택스트와 색 등을 변경할 수 있습니다.

    중간 실습

    모든 데이텀 디스플레이 유형을 비활성화합니다.

    1. '홈 탭 > 실행 그룹 > 결과'을 클릭합니다.

    2. '홈 탭 > 창 정의 그룹 > 열기'를 클릭합니다.

    3. Rod_ls를 선택하고 열기를 클릭합니다.

    4. 위와 같이 설정합니다. 확인 및 보기를 클릭합니다.

    5. '형식 탭 > 그래프 그룹 > 편집'을 클릭합니다.

    6. 탭에 따라 축 이름을 변경하거나 최댓값 최소값을 조정할 수 있습니다.

    7. 스타일 및 격자선을 조정하여 아래와 같이 그래프를 변경할 수 있습니다. '기본값으로 설정' 버튼이 있으니 이것 저것 test해보시기 바랍니다.

  • 결과 내보내기

    • '파일 탭 > Save as'를 클릭하면 결과를 다양한 형식으로 내보내기 할 수 있습니다.

    PVS - Creo view 용 파일로 View에서도 결과를 확인할 수 있습니다.

    html - 웹페이지 용 결과 내보내기입니다. 결과 형식에 따라 지원되지 않을 수 있습니다.

    ASCII - 그래프 형식의 텍스트 파일

    MS Excel - 그래프 형식의 텍스트 파일

    MPG, Avi - 동영상
